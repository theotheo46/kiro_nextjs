# NEW PROJECT IMPLEMENTATION

Implement Next.js based project for static web site implementation of calories calculator with the following features:

- user SHALL possibility to collect one or some products to the basket
- there is a list of products with icon or picture defined every product and amopunt of calories for 100 g for this product
- user can possibility to add or remove products from the basket
- basket recalculate its amount of calories every time when user asdd or delete the product
- make directory structure possible to usage with Github Pages hosting
